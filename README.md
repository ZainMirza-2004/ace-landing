# Ace Solutions

The public website for Ace Solutions, built with Next.js 15, React 19, and Tailwind CSS 4.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production checks

```bash
npm run lint
npm run build
```

## Deployment

The site is configured for deployment through Vercel. Contact form submissions are delivered to `support@acesols.com` through FormSubmit after the mailbox owner completes FormSubmit's one-time activation.
